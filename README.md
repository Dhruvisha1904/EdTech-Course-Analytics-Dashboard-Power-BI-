# EdTech-Course-Analytics-Dashboard-Power-BI-

📊 EdTech Course Analytics Dashboard

Project Overview :- 
=======>  Developed an interactive Power BI dashboard to analyze recorded lecture data from EdTech platforms. The dashboard provides insights into course performance across categories, languages, and instructors, helping identify learner engagement trends and high-demand content areas.

Key Insights :-
==> Identified top course categories and language preferences.
==> Analyzed how course duration, subtitles, and instructor ratings affect views.
==> Highlighted top instructors based on ratings.

Tools Used :- 
Power BI , Power Query , DAX

Outcome:-
Provided insights to help improve course strategy and learner engagement.
